# 🔐 Penetration Testing Labs

A collection of hands-on penetration testing labs demonstrating practical experience in cybersecurity and ethical hacking.

This repository documents structured penetration testing exercises conducted in controlled and authorized lab environments, covering various stages of the penetration testing lifecycle, including reconnaissance, enumeration, vulnerability assessment, exploitation, and post-exploitation.

## 🧪 Labs

| Lab | Target | Key Topics |
|---|---|---|
| Lab 01 | Metasploitable2 | Network Reconnaissance, Nmap, VSFTPD 2.3.4, Metasploit |
| Lab 02 | Windows 7 Enterprise | SMB Enumeration, MS17-010, EternalBlue, Post-Exploitation |
